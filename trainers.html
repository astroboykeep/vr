Совет рыбакам. Снасти

Катушки

Катушка рыболовная — это устройство для хранения лески на рыболовной снасти, представляет собой небольшую катушку с ручкой для сматывания лески. С помощью рыболовной катушки, установленной на удилище, можно забрасывать оснастку с приманкой на большое расстояние (спиннинг).

Крючки

При выборе крючков рыболов должен учитывать факторы, влияющие на результативность ловли:
- Активность рыбы
- Вес трофея
- Вид приманки

Поплавки

Бочкообразные поплавки отлично переносят среднее и сильное течения.
Плоский поплавок с длинным килем используются на сильном течении.
Поплавок с килем из металла позволяет выполнять дальние забросы.
